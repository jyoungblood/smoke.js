smoke.alert('this is a normal alert'); smoke.signal('this goes away after a sec'); smoke.confirm('use like a normal confirm?');

You can implement all of these the same way you'd use the js alert()...just put "smoke." in front of it.

The confirm() replacement, however, needs to be used just a little differently:

smoke.confirm('this is still cool, yeah?',function(e){
	if (e){
		alert('OK pressed');
	}else{
		alert('CANCEL pressed');
	}
});
